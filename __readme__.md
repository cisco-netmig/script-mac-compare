# MAC Compare

## Overview
**MAC Compare** is a tool used to compare and validate endpoint connectivity after a network cutover, ensuring devices successfully migrate from the old switch to the new access switch.

## Features
- Captures MAC address snapshots before and after migration.
- Compares pre-migration and post-migration MAC tables.
- Identifies missing or newly added devices.
- Generates detailed reports for validation.

## Migration Lifecycle
- Post-Migration

## Usage
**Create Snapshots:**  
- Enter a name.  
- Select "Pre" or "Post" type.  
- Capture MAC address data.  

**View and Manage Snapshots:**
- Right-click on a snapshot to:  
- View  
- Delete  
- Compare  
- Open snapshot folder  

**Compare MAC Tables:**  
- Select a "Pre" and "Post" snapshot.  
- Generate a comparison report to validate connectivity.  

**Export Reports:**  
- View the latest comparison report.  
- Open the results folder for further analysis.
